Golang基础--函数

2024-01-21 3 min read # 编程 # Golang

函数的声明和调用

函数声明包括函数名、参数列表和返回值类型列表。通过func 函数名(参数列表) (返回值类型列表)声明函数类型，然后在函数体中编写相应的代码。调用函数时只需使用函数名和相应的参数列表。示例代码如下：

package main

import fmt

func add(a, b int) int {
    return a + b
}

func main() {
    sum := add(3, 4)
    fmt.Println(sum) // 输出 7
}

匿名函数

匿名函数可以直接定义并调用，也可以赋给变量后再调用。这种灵活性使得匿名函数在需要临时使用功能时非常方便。示例代码如下：

package main

import fmt

func main() {
    add := func(a, b int) int {
        return a + b
    }

    sum := add(3, 4)
    fmt.Println(sum) // 输出 7
}

形式参数和实际参数

形式参数是函数定义中的参数，实际参数是调用函数时提供的值。函数调用时，提供的值必须与形式参数的数量、类型和顺序相匹配。示例代码如下：

package main

import fmt

func greet(name string) {
    fmt.Println(Hello,  + name)
}

func main() {
    greet(Alice) // 调用函数时，实际参数为 Alice
}

可变参数

可变参数允许函数接受任意数量的参数。在函数内部，可变参数被当做一个切片来处理，可以通过循环等方式进行处理。示例代码如下：

package main

import fmt

func sum(nums ...int) {
    total := 0
    for _, num := range nums {
        total += num
    }
    fmt.Println(total)
}

func main() {
    sum(1, 2, 3)       // 输出 6
    sum(1, 2, 3, 4, 5) // 输出 15
}

值传递

在Go语言中，函数的参数传递都是值传递，即函数的形参会被赋予实参的一份复制，对形参的操作不会影响到实参。示例代码如下：

package main

import fmt

func modify(num int) {
    num = 10
}

func main() {
    a := 5
    modify(a)
    fmt.Println(a) // 输出 5，a的值未受影响
}

引用传递

如果要在函数中修改变量的值，可以使用指针作为参数，实现引用传递。通过传递变量的地址来操作变量本身，从而达到修改变量值的目的。示例代码如下：

package main

import fmt

func modify(num *int) {
    *num = 10
}

func main() {
    a := 5
    modify(&a)
    fmt.Println(a) // 输出 10，a的值被修改为10
}

递归函数

递归函数是指在函数的定义中调用函数自身的一种函数。递归函数一般需要有一个结束条件，以避免无限递归。示例代码如下：

package main

import fmt

func factorial(n int) int {
    if n <= 1 {
        return 1
    }
    return n * factorial(n-1)
}

func main() {
    fmt.Println(factorial(5)) // 输出 120
}

Defer函数

defer语句用于延迟函数的执行，通常用于资源释放、文件关闭等操作。在函数返回之前，defer语句会将其后面的函数延迟执行。示例代码如下：

package main

import fmt

func main() {
    defer fmt.Println(World)
    fmt.Println(Hello,)
}
// 输出
// Hello,
// World

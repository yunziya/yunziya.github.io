Golang基础--流程控制

2024-01-21 2 min read # 编程 # Golang

GoTo

在Go语言中，goto语句将控制权转移到被标记的语句。语句通常与标签一起使用。以下是语句的用法示例：

package main

import fmt

func main() {
    i := 0
loop: // 定义标签
    if i < 5 {
        fmt.Println(i)
        i++
        goto loop // 转到标签loop处继续执行
    }
}

在上面的示例中，使用goto语句和标签loop来实现循环的功能。虽然语句在Go语言中不常用，但在某些特定情况下可能会用到。值得注意的是，在实际开发中，应避免滥用语句，以免影响代码可读性和可维护性。

IF判断

在Go语言中，使用if关键字进行条件判断。可以使用else if来添加额外的条件判断。

age := 18
if age < 0 {
    fmt.Println(年龄不能为负数)
} else if age < 18 {
    fmt.Println(未成年人)
} else {
    fmt.Println(成年人)
}

IF多重嵌套

可以在if语句中嵌套其他的if语句，也可以在if语句中嵌套switch语句。

x := 10
y := 20
if x > 0 {
    if y > 15 {
        fmt.Println(x大于0，y大于15)
    } else {
        fmt.Println(x大于0，y小于等于15)
    }
} else {
    fmt.Println(x小于等于0)
}

Switch判断

使用switch关键字进行多个条件判断。每个case可以包含多个条件。

day := 3
switch day {
case 1, 2:
    fmt.Println(工作日)
case 3, 4:
    fmt.Println(休息日)
default:
    fmt.Println(其他)
}

for循环

使用for关键字进行循环操作。可以使用break和continue关键字控制循环。

for i := 0; i < 5; i++ {
    if i == 2 {
        continue // 跳过本次循环
    }
    fmt.Println(i)
    if i == 3 {
        break // 结束循环
    }
}

range循环

使用range关键字遍历数组、切片、字符串、map等。

animals := map[string]string{cat: 猫, dog: 狗, bird: 鸟}
for key, value := range animals {
    fmt.Println(key, 的中文名是, value)
}

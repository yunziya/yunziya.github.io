Golang基础--运算符

2024-01-21 3 min read # 编程 # Golang

算术运算符

算术运算符用于执行基本的数学运算，如加法、减法、乘法和除法。

  • 加法：+，用于将两个数相加。
  • 减法：-，用于将一个数减去另一个数。
  • 乘法：*，用于将两个数相乘。
  • 除法：/，用于将一个数除以另一个数。
  • 求余：%，用于获取两个数相除的余数。
a := 10
b := 5
fmt.Println(a + b)  // 输出：15
fmt.Println(a - b)  // 输出：5
fmt.Println(a * b)  // 输出：50
fmt.Println(a / b)  // 输出：2
fmt.Println(a % b)  // 输出：0

关系运算符

关系运算符用于比较两个值的关系，返回布尔值（true或false）。

  • 等于：==，检查两个值是否相等。
  • 不等于：!=，检查两个值是否不相等。
  • 大于：>，检查左值是否大于右值。
  • 小于：<，检查左值是否小于右值。
  • 大于等于：>=，检查左值是否大于或等于右值。
  • 小于等于：<=，检查左值是否小于或等于右值。
x := 10
y := 5
fmt.Println(x == y)  // 输出：false
fmt.Println(x != y)  // 输出：true
fmt.Println(x > y)   // 输出：true
fmt.Println(x < y)   // 输出：false
fmt.Println(x >= y)  // 输出：true
fmt.Println(x <= y)  // 输出：false

逻辑运算符

逻辑运算符用于在布尔值之间执行逻辑运算，并返回布尔值作为结果。

  • 逻辑与：&&，当且仅当两个操作数都为true时，结果才为true。
  • 逻辑或：||，只要有一个操作数为true，结果就为true。
  • 逻辑非：!，用于取反一个布尔值。
a := true
b := false
fmt.Println(a && b)  // 输出：false
fmt.Println(a || b)  // 输出：true
fmt.Println(!a)      // 输出：false

位运算符

位运算符用于对整数的二进制位进行操作。

  • 按位与：&，将两个数的二进制位进行与操作。
  • 按位或：|，将两个数的二进制位进行或操作。
  • 按位异或：^，将两个数的二进制位进行异或操作。
  • 按位取反：^，将操作数的每个位取反。
a := 5      // 二进制：0101
b := 3      // 二进制：0011
fmt.Println(a & b)    // 输出：1  (0101 & 0011 = 0001)
fmt.Println(a | b)    // 输出：7  (0101 | 0011 = 0111)
fmt.Println(a ^ b)    // 输出：6  (0101 ^ 0011 = 0110)
fmt.Println(^a)       // 输出：-6

赋值运算符

赋值运算符用于为变量赋值。

  • 简单赋值：=，将右边表达式的值赋给左边的操作数。
  • 相加后赋值：+=，将左操作数与右操作数相加后再赋值给左操作数。
  • 相减后赋值：=，将左操作数与右操作数相减后再赋值给左操作数。
  • 相乘后赋值：=，将左操作数与右操作数相乘后再赋值给左操作数。
  • 相除后赋值：/=，将左操作数与右操作数相除后再赋值给左操作数。
x := 10
y := 5
x += y  // 相当于 x = x + y，此时 x 的值为 15
y *= 2  // 相当于 y = y * 2，此时 y 的值为 10

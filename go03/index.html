Golang基础--数据类型

2024-01-21 2 min read # 编程 # Golang

基本数据类型

Go语言的基本数据类型包括整型、浮点型、复数、布尔型、字符串和字符型。

// 整型
var intVar int = 10

// 浮点型
var floatVar float64 = 3.14

// 复数
var complexVar complex128 = 2 + 3i

// 布尔型
var boolVar bool = true

// 字符串
var stringVar string = Hello, world!

// 字符型
var runeVar rune = '中'

派生数据类型

Go语言的派生数据类型包括指针、数组、切片、字典、通道和结构体。

// 指针
var intVar int = 10
var intPtr *int = &intVar

// 数组
var arrayVar [3]int = [3]int{1, 2, 3}

// 切片
var sliceVar []int = []int{1, 2, 3}

// 字典
var mapVar map[string]int = map[string]int{a: 1, b: 2}

// 通道
var channelVar chan int = make(chan int)

// 结构体
type Person struct {
    Name string
    Age  int
}
var personVar Person = Person{Alice, 25}

其他数据类型

Go语言还有函数类型和接口类型作为其他数据类型。

// 函数类型
type Add func(a, b int) int
func addFunc(a, b int) int {
    return a + b
}

// 接口类型
type Shape interface {
    area() float64
}

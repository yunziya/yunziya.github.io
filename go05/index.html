获取键盘输入 in Go

在Go语言中，可以使用fmt包中的Scan和Scanln函数来获取键盘输入。

使用Scan函数获取键盘输入

Scan函数从标准输入中扫描文本，将以空格分隔的连续文本读入。输入以换行符结束。

示例代码：

package main

import fmt

func main() {
    var input string
    fmt.Println(请输入一个字符串：)
    fmt.Scan(&input)
    fmt.Println(你输入的是：, input)
}

运行示例代码后，程序会提示用户输入一个字符串，用户输入后按下Enter键，然后程序会打印出用户输入的字符串。

使用Scanln函数获取键盘输入

Scanln函数与Scan函数类似，不同之处在于它会在换行符之前停止扫描，因此可以用来获取一行文本输入。

示例代码：

package main

import fmt

func main() {
    var name string
    var age int
    fmt.Println(请输入您的姓名和年龄：)
    fmt.Scanln(&name, &age)
    fmt.Printf(您的姓名是：%s，年龄是：%d
, name, age)
}

在这个示例中，程序会提示用户输入姓名和年龄，用户输入后按下Enter键，然后程序会打印出用户输入的姓名和年龄。

以上是Go语言中使用Scan和Scanln函数获取键盘输入的方法。这些函数提供了一种简单的方法来从控制台获取用户输入。

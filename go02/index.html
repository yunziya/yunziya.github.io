Golang基础--常量

2024-01-20 2 min read # 编程 # Golang

概述

Go语言中的常量是一种在程序执行过程中数值不会发生改变的标识符。常量可以是布尔型、数字型（整数、浮点数）、字符型等。在Go语言中，常量的数值必须是在编译期就能够确定的。

常量的定义

在Go语言中，使用const关键字声明常量，并且需要在声明的同时对常量进行赋值。

// 定义一个整数常量
const num int = 10

// 定义一个字符串常量
const greeting string = Hello, World!

特殊常量

在Go语言中，有一些特殊的常量和预定义的常量，如true和false等。

// 布尔类型的常量
const (
    True  = true
    False = false
)

枚举类型

在Go语言中，常量可以作为枚举类型来使用，用于派生常量的值会在前一个常量值的基础上增加1。

// 枚举类型的常量
const (
    Monday = iota
    Tuesday
    Wednesday
    Thursday
    Friday
)

无类型常量

在Go语言中，常量可以是无类型的，并且有一些特殊的规则。无类型常量的类型可以根据上下文隐式转换。

// 无类型常量
const a = 10
const b = 3.14
const c = Hello

// 类型推断
var x int = a   // a被推断为int类型
var y float64 = b  // b被推断为float64类型
var z string = c   // c被推断为string类型
